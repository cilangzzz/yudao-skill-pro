# 表情系统接口

## 概述

表情系统是 IM 模块的辅助功能，提供系统表情包管理和用户私人表情管理能力。

### 业务定位

- 系统表情包由管理员维护，包含多个表情项
- 用户可收藏/上传私人表情
- 表情在消息中通过表情 ID 或名称引用

### 核心实体

| 实体 | 说明 |
|-----|------|
| FacePackDO | 系统表情包，包含名称、图标、排序、启用状态 |
| FacePackItemDO | 表情包中的单个表情项，包含图片 URL、名称、宽高 |
| FaceUserItemDO | 用户私人表情，包含图片 URL、名称、宽高 |

## 表情包接口

### 1. 获取启用的表情包列表

- **路径**: `GET /im/face-pack/list`
- **说明**: 获取所有已启用的系统表情包及其表情项
- **权限**: 需要用户登录
- **请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| - | - | - | 无额外参数 |

- **响应**: `CommonResult<List<FacePackDO>>` - 启用的表情包列表
- **实现逻辑**:
  1. 查询 `im_face_pack` 表中 `status = 启用` 的记录
  2. 关联查询每个表情包下的 `im_face_pack_item` 表情项
  3. 按 sort 字段排序返回

## 用户表情接口

### 2. 获取用户表情列表

- **路径**: `GET /im/face-user-item/list`
- **说明**: 获取当前用户的私人表情列表
- **权限**: 需要用户登录
- **请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| - | - | - | 无额外参数 |

- **响应**: `CommonResult<List<FaceUserItemDO>>` - 用户表情列表

### 3. 添加用户表情

- **路径**: `POST /im/face-user-item/create`
- **说明**: 添加私人表情
- **权限**: 需要用户登录
- **请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| url | String | 是 | 表情图片 URL |
| name | String | 是 | 表情名称 |
| width | Integer | 否 | 图片宽度 |
| height | Integer | 否 | 图片高度 |

- **响应**: `CommonResult<Long>` - 表情 ID

### 4. 删除用户表情

- **路径**: `DELETE /im/face-user-item/delete`
- **说明**: 删除私人表情
- **权限**: 需要用户登录
- **请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| id | Long | 是 | 表情 ID |

- **响应**: `CommonResult<Boolean>`

## 关键实现

- **FacePackService**: 表情包服务，管理表情包的 CRUD 和状态
- **FacePackItemService**: 表情项服务，管理表情包中的单个表情
- **FaceUserItemService**: 用户表情服务，管理用户的私人表情
- **图片存储**: 表情图片通过 FileApi 上传到文件存储服务
- **表情引用**: 消息中的表情通过表情名称或 URL 引用，存储在消息 content JSON 中
