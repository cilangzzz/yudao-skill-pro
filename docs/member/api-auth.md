# 会员认证 API

> C端用户认证相关接口，包含注册、登录、登出、社交登录等功能。

## 1. 概述

| 项目 | 说明 |
|------|------|
| Controller | `app/auth/AppAuthController` |
| Service | `MemberAuthService` |
| 路径前缀 | `/member/app/auth` |
| 认证方式 | 登录/注册接口使用 `@PermitAll`，无需登录 |

## 2. 接口清单

### 2.1 账号密码登录

| 项目 | 说明 |
|------|------|
| 接口 | 账号密码登录 |
| 说明 | 使用手机号 + 密码登录 |

**业务流程：**

1. 校验短信验证码（依赖 system 模块 `SmsCodeApi`）
2. 根据手机号查找会员用户
3. 校验密码（BCryptPasswordEncoder）
4. 生成 Token 返回
5. 记录最后登录 IP 和时间

**错误码：**

| 错误码 | 说明 |
|--------|------|
| 1_004_003_000 | 登录失败，账号密码不正确 |

### 2.2 手机号登录

| 项目 | 说明 |
|------|------|
| 接口 | 手机号 + 验证码登录 |
| 说明 | 使用手机号 + 短信验证码登录，支持自动注册 |

**业务流程：**

1. 校验短信验证码（依赖 system 模块 `SmsCodeApi`）
2. 根据手机号查找用户，不存在则自动注册
3. 生成 Token 返回

### 2.3 社交登录

| 项目 | 说明 |
|------|------|
| 接口 | 社交登录（微信、QQ 等） |
| 说明 | 通过第三方社交平台授权登录 |

**业务流程：**

1. 通过 `SocialClientApi` 获取社交用户信息
2. 根据社交平台 + openId 查找已绑定用户
3. 未绑定则自动创建会员用户
4. 生成 Token 返回

**依赖：**

- system 模块 `SocialClientApi`：社交登录授权

### 2.4 微信小程序登录

| 项目 | 说明 |
|------|------|
| 接口 | 微信小程序登录 |
| 说明 | 微信小程序 wx.login code 换取登录态 |

**业务流程：**

1. 使用 code 调用微信接口获取 openId / unionId
2. 根据 openId 查找已绑定用户
3. 未绑定则自动创建会员用户
4. 生成 Token 返回

### 2.5 会员注册

| 项目 | 说明 |
|------|------|
| 接口 | 会员注册 |
| 说明 | 手机号 + 验证码注册新会员 |

**业务流程：**

1. 校验手机号是否已注册
2. 校验短信验证码
3. 创建会员用户（密码使用 BCrypt 加密）
4. 发送 `MemberUserCreateMessage` MQ 消息
5. 生成 Token 返回

### 2.6 登出

| 项目 | 说明 |
|------|------|
| 接口 | 登出 |
| 说明 | 清除当前用户的 Token |

## 3. 跨模块依赖

| 模块 | API | 用途 |
|------|-----|------|
| system | SmsCodeApi | 短信验证码校验 |
| system | SocialClientApi | 社交平台授权登录 |

## 4. MQ 消息

| 消息 | 方向 | 时机 | 用途 |
|------|------|------|------|
| MemberUserCreateMessage | 生产者 | 用户注册成功后 | 通知其他模块处理新用户（如发放新人优惠券） |

## 5. 安全注意事项

- 密码使用 `BCryptPasswordEncoder` 加密存储，不可逆
- 登录接口使用 `@PermitAll` 标注，不需要认证
- Token 过期时间由 system 模块统一管理
- 短信验证码有频率限制，防止暴力破解
