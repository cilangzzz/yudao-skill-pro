# 草稿管理 API

> 对应 Controller：`MpDraftController` | 路径前缀：`/mp/draft`

## 功能说明

管理公众号图文消息草稿，支持草稿的创建、编辑、删除。草稿可发布为正式图文消息。

## 核心服务

| 方法 | 说明 |
|------|------|
| 创建草稿 | 创建图文消息草稿 |
| 编辑草稿 | 修改草稿内容 |
| 删除草稿 | 删除草稿 |
| 查询草稿列表 | 查询当前公众号草稿 |

## 与其他模块的关系

- 通过 `account_id` 关联 `mp_account`
- 草稿发布后通过 `MpFreePublishController` 管理发布

## 设计要点

- 草稿为微信侧管理，本地不做持久化
- 草稿内容为图文消息格式
- 草稿发布后变为正式图文消息
