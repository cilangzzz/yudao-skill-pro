# 发布能力 API

> 对应 Controller：`MpFreePublishController` | 路径前缀：`/mp/free-publish`

## 功能说明

管理公众号图文消息发布，支持从草稿发布为正式图文消息、查询发布状态、删除发布。

## 核心服务

| 方法 | 说明 |
|------|------|
| 发布 | 从草稿发布图文消息 |
| 查询发布状态 | 查询发布结果 |
| 查询发布列表 | 查询已发布图文列表 |
| 删除发布 | 删除已发布图文 |

## 与其他模块的关系

- 通过 `account_id` 关联 `mp_account`
- 发布内容来源于 `MpDraftController` 管理的草稿

## 设计要点

- 发布操作为微信侧管理，本地不做持久化
- 发布后图文消息对粉丝可见
- 支持发布状态查询
