# 统计域 API 文档

> 模块路径：`yudao-module-mall/yudao-module-statistics`

---

## 1. 交易统计

**Service**: `TradeStatisticsService`

### 1.1 功能说明

提供交易数据的汇总统计能力，包括订单量、销售额、客单价等核心指标。

### 1.2 统计维度

| 维度 | 说明 |
|------|------|
| 时间维度 | 按日/周/月统计交易数据 |
| 订单量 | 统计时间段内订单总数 |
| 销售额 | 统计时间段内支付金额总额 |
| 客单价 | 平均每笔订单支付金额 |

### 1.3 跨模块依赖

统计模块通过 `TradeOrderApi` 查询交易订单数据进行汇总。

---

## 2. 商品统计

**Service**: `ProductStatisticsService`

### 2.1 功能说明

提供商品维度的统计分析能力。

### 2.2 统计指标

| 指标 | 说明 |
|------|------|
| 商品销量排行 | 按销量排序的商品列表 |
| 库存预警 | 库存低于阈值的商品提醒 |
| 分类销售分布 | 各商品分类的销售占比 |

### 2.3 跨模块依赖

统计模块通过 `ProductSpuApi` / `ProductSkuApi` 查询商品数据进行汇总。

---

## 3. 数据来源

统计模块不直接访问业务数据表，而是通过以下方式获取数据：

| 数据来源 | API/方式 | 说明 |
|----------|---------|------|
| 交易数据 | `TradeOrderApi` | 查询订单列表进行聚合统计 |
| 商品数据 | `ProductSpuApi` | 查询商品信息和销量 |
| 定时任务 | 定时执行 | 定期汇总统计快照数据 |

---

## 4. 统计数据存储

统计结果存储在 `statistics` 子模块的数据表中，避免每次查询都实时聚合大量交易数据。
